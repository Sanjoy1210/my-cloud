# Assignment Challenge

## How to run code

1. Clone the repository
   ```sh
   git clone https://github.com/Sanjoy1210/my-cloud.git
   ```
2. Install dependencies

```sh
npm i
```

or

```sh
yarn
```

3. Run the project

```sh
npm run dev
```

or

```sh
yarn dev
```

Application should be available at http://localhost:8080/
